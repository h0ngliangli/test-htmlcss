通过设置body的width和 margin可以使得body居中。 

          body {
            width: 500px;
            margin: 0 auto;
            border: 1px solid #000;
          }

这是通过以下原理实现的：

  1. width: 750px;

    设置body的宽度,这样body就不会占据整个视口的宽度。

  2. margin: 0 auto;

    设置body的左右外边距为auto，这样body就会在视口中水平居中。
