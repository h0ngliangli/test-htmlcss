flex-grow属性

flex-grow是flex布局中的一个属性，用于控制子元素的放大比例。 它不会影响子元素的最小大小，而是决定它们在容器中的额外扩展空间。 它指定了一个比例因子，表示子元素和其他子元素在分配剩余空间时的 放大比例。

如果所有子元素的flex-grow属性都为1， 则它们将等分剩余空间。 如果一个子元素的flex-grow属性为2， 其他项目都为1，则前者占据的剩余空间将比其他项目多一倍。 它的默认值是0，表示不放大。

语法：

    .item {
        flex-grow: number; /* default 0 */
    }

number是一个无单位的数字，表示放大比例。

  • 0：不放大（默认值）
  • 1：等分剩余空间
  • 大于1：占据剩余空间的比例
