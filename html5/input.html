input

input 元素用于接收用户输入，常用于表单中。input 元素有多种类型，用于接收不同类型的输入，例如文本、密码、日期等。

基本用法

input 元素的基本用法如下：

    
<input type="text" placeholder="Enter your name">

常见类型

input 元素有多种类型，常见的类型如下：

  1. text: 默认类型，用于接收文本输入。 
    
<input type="text">
  2. password: 用于接收密码输入，输入内容会被隐藏。 
    
<input type="password">
  3. number: 用于接收数字输入。 
    
<input type="number">
  4. email: 用于接收邮箱地址输入。 
    
<input type="email">
  5. date: 用于接收日期输入。 
                
<input type="date">
  6. checkbox: 用于接收复选框输入。 
                
<input type="checkbox">
  7. radio: 用于接收单选框输入。 
                
<input type="radio">
  8. file: 用于接收文件输入。 
            
<input type="file">
  9. submit: 用于提交表单。 
            
<input type="submit"> <!-- 或者 --> <button type="submit">Submit</button>
  10. reset: 用于重置表单。 
            
<input type="reset"> <!-- 或者 --> <button type="reset">Reset</button>
  11. button: 用于触发自定义操作。 
            
<input type="button"> <!-- 或者 --> <button type="button">Click Me</button>

事件

input元素支持以下事件：

  1. 键盘事件
    • keydown: 当用户按下任意键时触发。
    • keyup: 当用户释放任意键时触发。
    • keypress: 当用户按下按键且该按键会产生字符输入时触发（但不再推荐使用，keydown 和 keyup 可以更好地替代它）。

    示例：

                
inputElement.addEventListener('keydown', function(event) {
    console.log('Key pressed: ', event.key);
});
  2. 鼠标事件
    • click: 当用户点击 input 元素时触发。
    • dblclick: 当用户双击 input 元素时触发。
    • mousedown: 当用户按下鼠标按钮时触发。
    • mouseup: 当用户释放鼠标按钮时触发。
    • mouseover: 当鼠标指针移动到 input 元素上时触发。
    • mouseout: 当鼠标指针移出 input 元素时触发。（会多次触发）。
    • mouseenter: 当鼠标进入输入框时触发（仅触发一次）。
  3. 焦点事件
    • focus: 当 input 元素获得焦点时触发。
    • blur: 当 input 元素失去焦点时触发。
    • focusin: 当 input 元素或其子元素获得焦点时触发。
    • focusout: 当 input 元素或其子元素失去焦点时触发。
  4. 输入事件
    • input: 当 input 元素的值发生变化时触发。
    • input: 当 input 元素的值发生变化时触发。
    • change: 当 input 元素的值发生变化并失去焦点时触发。
    • select: 当用户在输入框中选择文本时触发。
  5. 还有诸如剪切板、表单、拖放等事件，这次暂略。

属性

input元素支持以下属性：

  1. value: 设置或返回 input 元素的值。 
                
inputElement.value = 'Hello, World!';
console.log(inputElement.value); // 'Hello, World!'
  2. placeholder: 设置或返回 input 元素的占位符。 
                
<input type="text" placeholder="Enter your name">
  3. disabled: 设置或返回 input 元素是否禁用。 
                
inputElement.disabled = true;
console.log(inputElement.disabled); // true
  4. readonly: 设置或返回 input 元素是否只读。 
                
inputElement.readOnly = true;
console.log(inputElement.readOnly); // true
  5. checked: 设置或返回 input 元素是否被选中（仅适用于 checkbox 和 radio 类型）。 
                
inputElement.checked = true;
console.log(inputElement.checked); // true
  6. autofocus: 设置或返回 input 元素是否自动获得焦点。 
                  
<input type="text" autofocus>
  7. required: 设置或返回 input 元素是否为必填项。 
                  
<input type="text" required>
  8. pattern: 设置或返回 input 元素的验证规则。 
                  
<input type="text" pattern="[A-Za-z]{3}" title="Three letter word">

方法

input元素支持以下方法：

  1. focus(): 使 input 元素获得焦点。 
                
inputElement.focus();
  2. blur(): 使 input 元素失去焦点。 
                
inputElement.blur();
  3. select(): 选中 input 元素中的文本。 
                
inputElement.select();
