Asynchronous functions

In JavaScript, there are several asynchronous functions or APIs that allow for non-blocking operations, enabling code to continue executing while waiting for other tasks (like network requests, timers, or file operations) to complete. Here are some of the common asynchronous functions.

  1. setTimeout() and setInteval()

    setTimeout() and setInterval() are used to execute a function after a specified delay or at regular intervals, respectively. They are commonly used for animations, polling, and other time-based operations.

    • setTimeout() example: 
                  
                setTimeout(() => {
                console.log("Hello, world!");
                }, 1000);
    • setInterval() example: 
                  
                setInterval(() => {
                console.log("Hello, world!");
                }, 1000);
  2. Promises

    A Promise is a built-in JavaScript object representing the eventual completion (or failure) of an asynchronous operation. Promises are widely used in asynchronous programming to handle results when they are available.

    Example:

            
            const promise = new Promise((resolve, reject) => {
                setTimeout(() => {
                    resolve('Promise resolved');
                }, 1000);
            });

            promise.then(result => console.log(result)); // "Promise resolved" after 1 second
  3. Fetch API

    The Fetch API provides an interface for fetching resources (like JSON data, images, or files) across the network. It is a modern replacement for XMLHttpRequest and is commonly used in web applications to make HTTP requests.

    Example:

                
                fetch('https://api.example.com/data')
                    .then(response => response.json())
                    .then(data => console.log(data))
                    .catch(error => console.error(error));
