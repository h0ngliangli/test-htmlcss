::before 和 ::after 的含义

在 CSS 中，::before 和 ::after 是伪元素，它们用于在元素的内容前后插入内容。

::before 用于在元素内容的前面插入内容，而 ::after 用于在元素内容的后面插入内容。

伪元素是一种不在 DOM 中存在的元素，它们是通过 CSS 生成的。

::before 和 ::after 的应用场景

::before 和 ::after 可以用于在元素的内容前后插入内容，常用于添加装饰性的内容。

例如，你可以使用 ::before 和 ::after 为元素添加引号、图标、背景等。

::before 和 ::after 的语法

::before 和 ::after 的语法如下：


selector::before {
  content: "content";
}

selector 是 CSS 选择器，content 是要插入的内容。

你可以在 content 中插入文本、图片、图标等。

::before 和 ::after 的示例

示例 1：为元素添加引号

你可以使用 ::before 和 ::after 为元素添加引号。

例如，你可以为段落添加引号：


p::before {
  content: "“";
}

这样，每个段落的内容前面都会添加一个左引号。

示例 2：为元素添加图标

你可以使用 ::before 和 ::after 为元素添加图标。

例如，你可以为按钮添加一个下载图标：


button::after {
  content: url("download.png");
}

这样，每个按钮的内容后面都会添加一个下载图标。

总结

在 CSS 中，::before 和 ::after 是伪元素，它们用于在元素的内容前后插入内容。

你可以使用 ::before 和 ::after 为元素添加装饰性的内容，如引号、图标、背景等。
