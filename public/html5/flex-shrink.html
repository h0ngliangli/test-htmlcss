flex-shrink 的用法

语法：

flex-shrink: <number>;
  • <number>：一个非负数，表示元素缩小的比例。默认值是 1。
  • 如果为 0，则元素不会缩小。
  • 如果为正数（如 2），该元素会比其他元素缩小更多。

示例 1：默认缩小（flex-shrink: 1）

元素 1
元素 2

.flex-shrink-container {
  display: flex;
  width: 300px;
  border: 1px solid black;
}

.flex-shrink-child {
  width: 200px;
}

解释：

  • 父容器的宽度是 300px，而两个子元素的总宽度是 400px（2 × 200px）。
  • 由于 flex-shrink: 1 是默认值，两个子元素都会按 相同比例 缩小，以适应容器的宽度。

示例 2：设置不同的缩小比例

元素 1
元素 2

.flex-shrink-container {
  display: flex;
  width: 300px;
  border: 1px solid black;
}

.flex-shrink-child1 {
  width: 200px;
  flex-shrink: 1;
}

.flex-shrink-child2 {
  width: 200px;
  flex-shrink: 2;
}

解释：

  • 容器总宽度是 300px，但两个子元素的总宽度是 400px（2 × 200px）。
  • 子元素 1 的缩小比例是 1，子元素 2 的缩小比例是 2。
  • 由于元素 2 的 flex-shrink 值较大，因此它会比元素 1 缩得更多。

示例 3：不允许元素缩小（flex-shrink: 0）

元素 1
元素 2

.flex-shrink-container {
  display: flex;
  width: 300px;
  border: 1px solid black;
}

.flex-shrink-child1 {
  width: 200px;
  flex-shrink: 0;
}

.flex-shrink-child2 {
  width: 200px;
  flex-shrink: 1;
}

解释：

  • 子元素 1 的 flex-shrink 值是 0，所以它不会缩小，即使容器空间不足。
  • 子元素 2 会缩小，以适应容器的总宽度。

flex-shrink 的计算公式

当容器中的元素需要缩小时，每个元素的缩小宽度由以下公式计算：

元素缩小宽度 = (原始宽度 × flex-shrink) / 所有元素的 flex-shrink 总和

常见问题：

  • flex-shrink 只在空间不足时起作用。
  • flex-shrink 与 flex-grow 的关系：
    • flex-shrink：定义当空间不足时，元素如何缩小。
    • flex-grow：定义当有剩余空间时，元素如何扩展。

小结

  • flex-shrink 控制当容器空间不足时，元素缩小的比例。
  • flex-shrink: 0 禁止元素缩小。
  • 较大的 flex-shrink 值会让元素比其他元素缩得更快。
  • 默认情况下，所有元素的 flex-shrink 值为 1，即按相同比例缩小。
  • 比如本站，如果nav中不加flex-shrink: 0，那么nav会随着搜索框的增大而缩小，这样不但不好看，而且内容显示不全。
