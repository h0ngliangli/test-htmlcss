padding vs margin

  • 当你希望元素内部的内容 与其边框之间留出空间时，使用 padding。例如，按钮中的文字和按钮边框之间的距离。
  • 你希望元素之间 保持间距时，使用 margin。例如，两个段落之间的间距。

CSS 选择器的使用：什么时候用 class、id、> ？

  • class 是 CSS 中最常用的选择器，因为它可以被 多次复用，适用于多个元素。
  • id 是一种 唯一选择器，只能在整个页面中用于 一个元素。它的权重比 class 高。
  • 优先使用 class 而不是 id. 避免过度使用 id 来定义样式，以保持样式的一致性和可维护性。尽量使用 class。
  • 避免过度使用嵌套选择器. 深层嵌套的选择器（如 .container .parent .child）会降低代码的可维护性。尽量保持选择器简洁。
  • 避免给选择器增加太多权重。优先使用简单的 class 选择器，避免大量使用 !important。
