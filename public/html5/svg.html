svg

SVG(Scalable Vector Graphics) 是一种基于 XML 的图形格式，用于创建可缩放的矢量图形。

SVG 可以通过以下几种方式使用：

  • 作为 HTML 元素内联使用
  • 作为 HTML 元素的背景
  • 作为 CSS 的背景
  • 作为图片使用

SVG 的优点：

  • 无限缩放：SVG 是矢量图形，因此可以任意缩放而不会失去清晰度。无论在小屏幕上还是大屏幕上，它都能保持高质量的显示，适合高清显示设备（如 Retina 屏幕）。
  • 文件体积小：对于简单或中等复杂的图形，SVG 文件的体积通常比同样尺寸的位图小。
  • 可编辑和可搜索：由于 SVG 是基于 XML 的文本格式，图形可以直接在文本编辑器中编辑，无需图像处理软件。同时，SVG 内容可以被搜索引擎索引，提高网页的 SEO 性能。
  • 可编程和动态交互：SVG 可以与 CSS 和 JavaScript 结合使用，允许动态样式变化、动画、事件绑定等。可以为 SVG 图形中的部分元素添加点击、悬停等交互效果。
  • 可压缩
  • 样式和动画支持：SVG 图形可以通过 CSS 进行样式化，并支持 SMIL（Synchronized Multimedia Integration Language）或 CSS 动画 来为图形添加动画效果。

SVG 的缺点：

  • 复杂图形的绘制和编辑相对困难
  • 不适合处理大量的位图数据
  • 不适合处理复杂的动画效果

总结，SVG 是非常强大的工具，尤其适用于制作矢量图形、图标和动画，但对于复杂、细节丰富的图片，位图格式（如 JPEG 或 PNG）可能更适合。

用法：

  1. 直接在 HTML 中嵌入svg元素。 
          
        <!-- 嵌入简单的 SVG 图形 -->
        <svg width="100" height="100">
            <!-- 定义一个红色的圆 -->
            <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
        </svg>
      
    
    • <svg width="100" height="100">
      定义一个 100x100 像素的画布。
    • circle：绘制一个圆，其中 cx 和 cy 表示圆心的坐标，r 表示半径，stroke 和 fill 分别设置边框和填充色。
  2. 通过img元素加载外部 SVG 文件 
          
        <img src="circle.svg" alt="Circle" />
  3. 使用object、iframe 或 embed 标签嵌入 SVG 
          
        <object data="circle.svg" type="image/svg+xml"></object>
        <iframe src="circle.svg"></iframe>
        <embed src="circle.svg" type="image/svg+xml" />
  4. 使用 CSS 设置背景为 SVG 
          
        <div style="background-image: url('circle.svg');"></div>
