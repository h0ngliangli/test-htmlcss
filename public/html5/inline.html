Inline Elements

Inline elements are elements that do not start on a new line and only take up as much width as necessary. They are used within block-level elements or other inline elements. Some common inline elements include:

  • span
  • a
  • strong
  • em
  • img
  • br
  • code

Inline elements can be nested within other inline elements or block-level elements, but they cannot contain block-level elements.

Example:


<p>This is an example of a <strong>paragraph</strong> with inline elements.</p>

Block-Level Elements

Block-level elements are elements that start on a new line and take up the full width available. They are used to create the structure of a webpage and can contain other block-level elements, inline elements, or text content. Some common block-level elements include:

  • div
  • p
  • pre
  • h1 to h6
  • ul and ol
  • li
  • table

Block-level elements can be nested within other block-level elements, but they cannot be nested within inline elements.

Example:

    
        <div>
            <h1>This is a heading</h1>
            <p>This is a paragraph</p>
        </div>

One special case is the p element, which is a block-level element that represents a paragraph of text. It automatically adds space before and after the content, creating a separation between paragraphs. And it can't contain block-level elements, only inline elements.

Example:

    
        <p>This is a paragraph.
            <pre>
            This is preformatted text. <!-- This is wrong but browsers can tolerate it. -->
            </pre>
        </p>

Inline-Block Elements

Inline-block elements are a combination of inline and block-level elements. They start on a new line like block-level elements, but they only take up as much width as necessary like inline elements. They can contain other inline or block-level elements.

Some common inline-block elements include:

  • img
  • button
  • input
  • select
  • textarea

Inline-block elements can be used to create elements that need to be displayed inline but also require block-level styling, such as buttons or form elements.

Example:

    
        <button>Click Me</button>
